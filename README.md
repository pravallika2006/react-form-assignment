# Registration Form Assignment

This is a React assignment demonstrating a registration form with validation and routing.

## Features
- **Form Validation:** Checks for empty fields, valid email format, and password length.
- **Show/Hide Password:** Users can toggle password visibility.
- **Conditional Rendering:** Error messages appear in red only when inputs are invalid.
- **React Router:** Redirects to a "Details Page" upon successful submission.
- **Responsive Design:** Works on mobile and desktop using standard CSS.

## How to Run
1. Clone the repository.
2. Install dependencies:
   npm install
3. Start the application:
   npm start
